General Project Goals:

-develop GUI including a keyboard, buttons for waveform selection, and volume
-synthesize waveforms in realtime and output sounds to pc audio system
-testbed/prototype for future developments


Project Requirements:

-GUI built with pyQT and contains styling elements
-synth plays one note at a time, notes can be held using mouse button
-sound generated should be relatively smooth (little to no artifacts)
-can select between sine and square waveforms
